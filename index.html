Instalar Windows Server desde el medio de instalación

Aprenda a crear una unidad USB o un DVD de arranque que se puedan usar para instalar Windows Server en sus dispositivos. La creación de un medio de instalación de Windows Server es un paso crucial para configurar un nuevo servidor o actualizar uno existente. Este artículo tiene como objetivo guiarlo sobre cómo crear medios de instalación e instalar Windows Server utilizando las opciones de instalación Server Core o Desktop Experience.

Prerrequisitos

Los medios de instalación para la versión de destino de Windows Server se pueden obtener de los canales de fabricantes de equipos originales (OEM), minoristas, suscripción a Visual Studio y del Centro de servicios de licencias por volumen (VLSC). El dispositivo tampoco debe ejecutarse en Azure. Si desea instalar Windows Server en Azure, consulte Inicio rápido: Crear una máquina virtual de Windows en el portal de Azure . Se requieren los siguientes elementos antes de instalar Windows Server:

Hardware

  1. Su dispositivo debe cumplir o superar los requisitos mínimos de hardware. Para obtener más información, consulte Requisitos de hardware para Windows Server.
  2. Una unidad flash USB de al menos 8 GB.
  3. Una unidad óptica capaz de grabar un disco DVD+/-RW de doble capa.
  4. Un DVD de doble capa de 8,5 GB.

Software

  1. Determine qué versión de Windows Server es la adecuada para su entorno. Para obtener más información, consulte Comparación de ediciones de Windows Server.
  2. Asegúrate de tener una clave de producto o una licencia de suscripción válidas para tu producto. Las claves de producto y los métodos de activación pueden variar según el canal de distribución del que hayas recibido los medios de Windows Server, como un programa de licencias comerciales, un minorista o un OEM.

Crear una unidad flash USB de arranque

En general, la creación de una unidad USB de arranque se puede realizar mediante la utilidad diskpart.exe . Como alternativa a la preparación manual de la unidad USB, los usuarios pueden ejecutar el siguiente script de PowerShell para que la unidad USB sea de arranque. El archivo ISO de instalación de Windows Server debe estar montado antes de ejecutar este script. Para montar el archivo ISO, realice los siguientes pasos:

  1. Localice el archivo ISO de instalación de Windows Server.
  2. Haga clic derecho en su archivo ISO y seleccione Montar.

Al montar el archivo ISO se crea una unidad óptica virtual, con su propia letra de unidad, donde los usuarios pueden migrar el contenido ISO a su unidad USB.
